# book-club 設計規範（MASTER）

> 方向：**書房夜讀風**——Andy 於 2026-07-04 核准（TG「風格先1」）。深夜檯燈下讀實體書的感覺：暖、靜、不刺眼。任何頁面色/字/距只准引用本檔 token。

## 色彩（CSS custom properties）
### 日間（預設 light）
- `--paper: #F7F1E3`（暖米紙）
- `--ink: #2B2118`（深咖啡墨）
- `--ink-soft: #6B5D4F`（次要文字）
- `--accent: #8C5A2B`（皮革棕，連結/強調）
- `--rule: #E4D9C3`（分隔線）
- `--card: #FFFBF0`（引文/金句底）
### 夜間（prefers-color-scheme: dark ＋手動切換）
- `--paper: #12160F`（墨綠黑）
- `--ink: #E9E2D0`（暖米字）
- `--ink-soft: #A79E8B`
- `--accent: #E8B84B`（暖檯燈黃）
- `--rule: #2A2F24`
- `--card: #1A1F15`

### 主題書封色（圖書館目錄頁書封用；日夜共用，深色上配 --paper-tone 文字）
- `--theme-growth: #5B6E4F`（自我成長＝苔綠）
- `--theme-career: #3E5C76`（職場成長＝墨藍）
- `--theme-people: #96604A`（人際關係＝赭紅）
- `--theme-logic: #6B5B7B`（邏輯思考＝暮紫）
- 書封文字一律 `#F5EFE0`（暖米，四色上對比皆 ≥4.5:1）

## 字體
- 標題與內文：`"Noto Serif TC", "Songti TC", serif`（Google Fonts，僅載 400/700 兩字重）
- UI 小標（日期、主題 chip、閱讀時間）：`"Noto Sans TC", sans-serif`
- 內文 19px／行高 1.9／段距 1.4em；手機 18px。標題階層 clamp：h1 1.9~2.4rem、h2 1.35rem、h3 1.1rem
- 中文與英數之間半形空格；標點懸掛不強求

## 版式
- 單欄置中，內文欄寬 max 42rem；頁面左右 padding clamp(1.25rem, 5vw, 2.5rem)
- 縱向節奏 8 的倍數；區塊間 48px、段落間 24px
- 金句區：`--card` 底＋左側 3px `--accent` 邊線＋Serif 斜體不用（中文斜體醜，用字重）
- 討論題區：卡片＋「回 TG 聊聊」按鈕（連到 t.me；tg://不穩不用）
- 進度條：頁面頂部細線（scroll progress，3px，`--accent`），JS 10 行內
- 圓角統一 10px；陰影禁用（平面層次，用邊線和色差）

## 動效
- 只有 hover 連結下劃線淡入與夜間切換的 color transition 200ms；respect prefers-reduced-motion
- 禁：進場動畫、視差、打字機效果

## 2026-07-04 手法移植附錄：「書桌上的紙」（參考 letusibiza.com，Andy 核准方案 1，只搬手法不搬色）

1. **畫布層（桌面）**：`body` 背景改為桌面色 token——日間 `--desk: #43362A`（溫暖深咖啡桌面），夜間 `--desk: #080A06`（比紙更深一階的深夜）。桌面上永遠看得到。
2. **紙張層**：頁面內容包在 `.sheet`——`background: var(--paper)`、`border-radius: 24px`、四周 `margin: clamp(12px, 2.5vw, 28px)`（露出桌面邊框）、內部維持原 42rem 內容欄。首頁與書摘頁都套。頁腳可留在桌面上（紙外，文字用 `--paper` 的 70% 透明色）。
3. **超大細字重標題**：display 級（站名、書摘書名、藏書統計數字）→ `font-weight: 400`＋行高 1.05＋字級放大（站名 clamp(2.6rem,7vw,4.2rem)、書名 clamp(2.2rem,6vw,3.4rem)）。**內文 h2/h3 保留 700**（中文長文掃讀性優先，這是刻意偏離參考站的點）。
4. **單字體紀律不變**（Noto Serif TC 內文＋Noto Sans TC UI 小標），層級靠尺寸與留白，不加新字體。
5. **按鈕/chip 成對**：主要動作＝實心 `--accent` 膠囊（radius 999px）；次要/篩選＝1.5px 細框膠囊，選中轉實心。
6. **口語 hook**：首頁站名下加一句大字 hook「今晚，翻開一本就好。」（原副標降為小字）。
7. 圓角家族更新：紙張 24px、書封 6px、按鈕/chip 999px、其他卡 10px。陰影仍然全站禁用；紙張與桌面的層次靠色差，不靠陰影。

## 2026-07-04 深夜版型改版：圖書館卡片目錄（Andy 嫌書封牆醜＋整體詭異後選定方案 2）

首頁放棄書封牆，改「卡片目錄」；書桌×紙張畫布與字體規則不變。書摘內頁不動。
1. **刊頭（2026-07-04 深夜終版，Andy 嫌俗氣後改）**：站名「**拾頁**」（display 400、letter-spacing 0.14em、clamp(3rem,10vw,4.4rem)）→ 3.2rem 金色髮絲線（--accent 1px）→ 一行安靜資訊「每晚七點・一本書的深度導讀・已上架 N 本」。**不放標語**（「今晚翻開一本就好」已刪，slogan 感=俗氣來源）。
1b. **收藏系統**：卡片右上＋內頁頂部書籤鈕（SVG 細線 44px，實心=已收藏，localStorage key `yedu-favs`）；篩選列「我的收藏」chip；控件（搜尋+chips）在過往藏書 ≥1 時出現。
2. **今晚主打卡**：最新一本（date=今天或最近）放大版目錄卡——「今晚的書」accent 小 chip＋迷你書封（96px 寬）＋書名＋作者＋完整 hook＋實心膠囊「開始閱讀」。
3. **目錄卡（過往藏書）**：橫式純排印卡（--card 底、1px --rule 框、**左緣 4px 主題色書籤條**、radius 10px）。內容＝索引行小字「No.001・自我成長・M/D」→ 書名（serif 700 1.18rem）→ 作者小字 → hook 最多兩行截斷 → 底行「閱讀 N 分鐘」＋accent「開始閱讀 →」。**迷你書封已依 Andy 指示移除（2026-07-04 深夜，「版面不錯但把書拿掉」）——主題色識別只靠書籤條，不再畫假書。**今晚主打卡同樣無書封＋書籤條。
4. **即將上架＝安靜列表**：一行一本「7/5・深度工作力 — Cal Newport」（--ink-soft、無連結、無卡片），區塊標題小。不再用半透明幽靈卡。
5. **控件晚出**：主題篩選 chips 與搜尋框只在已上架 ≥5 本時 build 進頁面（build.mjs 條件輸出），藏書少時頁面只有內容。
6. 迷你書封是目錄卡的一部分，尺寸小、細節靠書脊與髮絲框——不做大幅書封牆。
- 禁 emoji 當結構 icon；主題標籤用文字 chip（細框）
- 禁裸 hex（一律 var(--token)）；禁漸層、毛玻璃
- 內文 ≥18px、對比 ≥4.5:1、觸控目標 ≥44px、手機無橫向捲動

## 2026-07-05 新增：日期篩選「月份 → 每日書單」（Andy 於 2026-07-05 TG 核准方案 1）

需求＝除書名/作者搜尋外，能「點某月看那月每一天的書單」。方向 1（月份標籤→每日書單）核准，先提規劃 Andy 點頭「開工」。

1. **入口＝月份 chip 列**：放在搜尋框與主題 chips 之間，多一排月份鈕「全部｜七月｜八月…」，依實際有書的月份自動生成、由近到遠。月份多時整排 `overflow-x:auto` 可左右滑（比照 `.myshelf-strip` 隱藏捲軸、scroll-snap）。沿用 `.filter-chip` 細框膠囊樣式，選中轉實心。預設「全部」＝現有首頁不變。
2. **模式切換（全部 vs 某月）**：點某月＝進「月視圖」——**收起** 今晚主打卡／我的書櫃／金句集／藏書目錄／即將上架，只顯示 `#month-view`；點「全部」還原。純前端 toggle `hidden`，無跳頁。
3. **月視圖版型**：頂部小標「七月・共 N 本」；下面一天一列、**由月初排到月底（升冪，像翻行事曆往下滑）**。列＝左側日期方塊（大數字日＋小字星期，`font-variant-numeric:tabular-nums` 對齊）＋左緣 4px 主題色書籤條＋書名（serif 700）＋作者·主題（sans soft）＋一行 hook（clamp 1 行）＋右側動作。
4. **一天三狀態**：
   - 已上架（date ≤ 今天）：整列 `<a>` 可點進書摘頁，hover translateY(-2px)（同目錄卡），右側「讀 →」。
   - 今天：多一個 accent「今晚」小標籤。
   - 未上架（date > 今天）：`opacity:~0.55`、日期方塊與字轉柔、右側「即將上架」細框膠囊、**非連結不可點**（`aria-disabled`）——保留每晚 7 點解鎖儀式，但書名＋hook 先當預告公開（與現有「即將上架」清單一致，titles 本就公開）。
5. **與搜尋/主題疊加**：月視圖內搜尋框＋主題 chips 仍生效（AND），都不中顯示「七月裡沒有符合的書」空狀態。
6. **資料**：build 需把「未上架書」的最小資料（date/slug/title/author/theme/hook）也送進前端（新 `window.__calendar` 全書陣列＋locked 旗標）；現有 `window.__books`（僅已上架，供書櫃/搜尋）不動。
7. 色/字/距/圓角全引用既有 token；日期方塊用主題色書籤條識別、不畫假書封；禁 emoji icon、禁漸層；觸控 ≥44px、respect reduced-motion。
